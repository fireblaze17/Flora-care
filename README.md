# FloraCare - Plant Disease Identification & Care Management System

A comprehensive platform integrating computer vision and web technologies to monitor, detect, and ensure the health of various plant species. The system leverages AI models for disease identification and focuses on efficient plant care management.

## Project Overview

**Key Highlights:**
- Provides accurate detection of plant diseases using well-trained CNN models.
- Manages plant care schedules with a full-stack web application for personalized plant tracking.
- Prioritizes real-time updates and actionable insights for plant disease treatment.

### Technical Features:
1. **AI-Driven Disease Detection**:
    - Deep learning models designed for image-based plant health monitoring.
    - Trained on various datasets to classify diseases like fungal infections, bacterial spots, viral ailments, and pest damage.

2. **Full-Stack Web Application**:
   - Combines Flask as a backend with MySQL for database management.
   - Implements plant addition, editing, and health management features.

3. **Database**:
   - Tracks health metrics of registered plants and keeps user logs.

4. **Notifications & Alerts**:
   - Integrates Firebase for push notifications about timely watering and disease conditions.

5. **Multi-device Friendly**:
   - Responsive design with seamless functionality on both mobile and desktop platforms.

## Insights Gained:
### Technical:
- Enhanced understanding of the convolutional neural networks used in image classification tasks.
- Integration of open-source web-based backends like Flask with modern database communication.
- Automatic notification pipelines using Firebase.

### Practical Skills:
This project instilled new ways to:
- Pre-process agricultural datasets to highlight problematic leaves and plant regions.
- Secure user data and ensure scalable and efficient database architectures.
- Deliver web development interfaces that preserve usability even under endpoint stress testing.
